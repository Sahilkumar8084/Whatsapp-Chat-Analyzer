



# 📊 WhatsApp Chat Analyzer

A powerful web-based application that analyzes your WhatsApp chats and provides meaningful insights through interactive visualizations.

🚀 **Live App:** https://whatsapp-chat-analyzer-web.streamlit.app/
---

## ✨ Features

- 📈 **Top Statistics**
  - Total messages
  - Words count
  - Media shared
  - Links shared

- 👤 **Most Active Users**
  - Identify who chats the most in a group

- 🗓️ **Timeline Analysis**
  - Daily & monthly message trends

- 🔥 **Activity Heatmap**
  - Analyze most active days and hours

- ☁️ **Word Cloud**
  - Visualize most frequently used words

- 😀 **Emoji Analysis**
  - Most used emojis
  - Emoji frequency insights

- 💬 **Sentiment Analysis**
  - Detect positive, negative, and neutral messages

- 🔍 **User-wise Analysis**
  - Filter and analyze individual participants

---

## 📂 How It Works

1. Export your WhatsApp chat:
   - Go to WhatsApp → Chat → More → Export Chat
   - Choose **Without Media**

2. Upload the `.txt` file to the app

3. Get instant insights & visualizations 📊

---

## 🛠️ Tech Stack

- **Python**
- **Streamlit** (Frontend UI)
- **Pandas** (Data processing)
- **Matplotlib / Seaborn / Plotly** (Visualization)
- **WordCloud**
- **Regex (re)** for parsing chats

## ⚙️ Installation (Run Locally)

# Clone the repository
git clone https://github.com/your-username/whatsapp-chat-analyzer.git

# Navigate to project folder
cd whatsapp-chat-analyzer

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
